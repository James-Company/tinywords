# 20 RELEASE CHECKLIST

이 문서는 TinyWords MVP 릴리즈 실행 체크리스트 SSOT이다.  
목표는 출시 전후 품질/보안/운영 리스크를 줄이고, Go/No-Go 결정을 일관된 기준으로 수행하는 것이다.

---

## 1) 사용 방법

- 릴리즈마다 본 문서를 복제해 실행 로그로 사용한다.
- 각 항목은 `완료자 / 일시 / 근거 링크`를 남긴다.
- Blocker/Critical 이슈가 하나라도 열려 있으면 출시 금지.

실행 상태 표기:
- `[ ]` 미완료
- `[~]` 진행중
- `[x]` 완료
- `[!]` 차단 이슈
