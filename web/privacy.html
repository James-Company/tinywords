개인정보처리방침

최종 업데이트: 2026년 2월 15일

TinyWords(이하 "서비스")는 회원의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 회원의 개인정보가 어떻게 수집·이용·보관·파기되는지 안내합니다.

1. 수집하는 개인정보

1.1 회원가입 시 필수 수집

항목수집 목적
이메일 주소회원 식별, 로그인, 이메일 인증, 비밀번호 재설정
비밀번호 (암호화 저장)본인 인증

1.2 소셜 로그인 시 수집

항목수집 목적

이메일 주소회원 식별, 알림
프로필 이름 (선택)서비스 내 표시명

1.3 서비스 이용 중 자동 수집

항목수집 목적
학습 기록 (학습 항목, 회상 결과, 스트릭)복습 스케줄 생성, 학습 진도 제공
사용자 작성 문장AI 문장 코칭 제공
음성 녹음 파일발음 피드백 제공 (로컬 저장 기본)
앱 설정 (하루 학습량, 레벨 등)맞춤형 학습 경험 제공
접속 기록, 기기 정보, 타임존서비스 안정성, 오류 대응

2. 개인정보의 이용 목적

  • 회원 가입 및 관리: 본인 확인, 계정 관리, 부정 이용 방지
  • 서비스 제공: 일일 학습, 복습 큐, AI 콘텐츠 생성, 발음 피드백
  • 서비스 개선: 이용 통계 분석, 오류 진단, 기능 개선
  • 안내 및 공지: 서비스 변경사항, 약관 변경 안내

3. 개인정보의 보유 및 이용 기간

구분보유 기간근거
회원 정보회원 탈퇴 시 즉시 파기회원 요청
학습 기록회원 탈퇴 시 즉시 파기회원 요청

접속 기록3개월통신비밀보호법

4. 개인정보의 제3자 제공

회사는 원칙적으로 회원의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다.

  • 회원이 사전에 동의한 경우
  • 법령에 의해 요구되는 경우

5. 개인정보의 위탁

수탁자위탁 업무
Supabase (미국)회원 인증, 데이터 저장
OpenAI (미국)AI 학습 콘텐츠 생성, 문장 코칭

위탁 시 개인정보가 안전하게 처리되도록 계약을 통해 관리·감독합니다.

6. 개인정보의 국외 이전

서비스 제공을 위해 다음과 같이 개인정보가 국외로 이전될 수 있습니다.

이전 대상이전 국가이전 항목이전 목적
Supabase Inc.미국이메일, 학습 데이터인증 및 데이터 저장
OpenAI Inc.미국학습 설정, 사용자 문장 (비식별)AI 콘텐츠 생성

7. 개인정보의 파기

  1. 보유 기간 경과 또는 처리 목적 달성 시 지체 없이 파기합니다.
  2. 전자적 파일: 복구 불가능한 방법으로 영구 삭제합니다.
  3. 로컬 저장 데이터(음성 등): 기기에서 삭제됩니다.

8. 회원의 권리 및 행사 방법

회원은 언제든지 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 정정·삭제 요청
  • 개인정보 처리 정지 요청
  • 회원 탈퇴 (설정 화면에서 직접 가능)

요청은 서비스 내 설정 또는 이메일(support@tinywords.today)을 통해 가능하며, 지체 없이 처리합니다.

9. 개인정보의 안전성 확보 조치

  • 비밀번호 암호화 저장 (bcrypt)
  • 모든 통신 TLS(HTTPS) 암호화
  • 접근 권한 최소화 및 관리
  • API 키 서버 측 보관, 클라이언트 노출 금지
  • 음성 파일 로컬 우선 저장, 외부 전송 기본 비활성
  • 로그 내 민감정보 마스킹 처리

10. 쿠키 및 자동 수집 장치

서비스는 인증 세션 유지를 위해 브라우저 로컬 스토리지를 사용합니다. 분석·추적 목적의 쿠키는 사용하지 않습니다.

11. 개인정보 보호책임자

  • 담당: TinyWords 운영팀
  • 이메일: privacy@tinywords.today

개인정보 관련 문의, 불만, 피해 구제는 위 연락처로 접수해주시면 신속히 처리하겠습니다.

12. 방침의 변경

본 방침이 변경되는 경우, 변경 사항을 서비스 공지사항 또는 이메일을 통해 안내합니다. 변경된 방침은 공지한 날로부터 7일 후에 효력이 발생합니다.

본 개인정보처리방침은 2026년 2월 15일부터 시행됩니다.
